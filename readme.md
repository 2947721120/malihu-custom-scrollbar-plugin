malihu自定义滚动条插件
================================

高度可定制的自定义滚动条的jQuery插件。功能包括: 
* 垂直或水平滚动条（S）
* 可调的滚动动量
* 鼠标滚轮，键盘和触摸支持
* 准备通过CSS使用主题和定制
* 支持RTL方向
* 为滚动条的功能完全控制选项参数
* 触发动作，如滚动，更新，破坏等方法。
* 用户定义的回调函数
* 可选/可搜索内容

#### 安装

Bower: `bower install malihu-custom-scrollbar-plugin` 

npm: `npm install malihu-custom-scrollbar-plugin`
